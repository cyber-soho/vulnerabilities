Part B: Patch Deployment

The core operational phase. Follow the lifecycle: Snapshot -> Verify Integrity -> Deploy -> Log. Choose your target OS below to see specific commands.

1. Establish Baseline

Record current state before touching anything.

wmic qfe list brief systeminfo | findstr /B /C:"OS Version"
uname -r apt list --upgradable
‚ö†Ô∏è Take VM Snapshot Now!

2. Verify Integrity

Ensure patches are authentic.

Calculate SHA256 of .msu file:

Get-FileHash -Path "patch.msu" -Algorithm SHA256

Package managers (apt/dnf) use GPG keys automatically. Verify keys:

apt-key list

3. Install Updates

Apply the fix and reboot if needed.

wusa.exe patch.msu /quiet /norestart

OR (Cumulative)

Install-WindowsUpdate -AcceptAll
sudo apt update && sudo apt upgrade -y

RedHat/CentOS:

sudo dnf update --security -y
